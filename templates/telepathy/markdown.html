Markdown

DocHub utilise Markdown pour mettre en forme les discussions et les descriptions des documents.

Le Markdown c'est quoi ?

Markdown est une convention permettant une mise en page structurée grâce à des annotations simples dans le texte. Vous pouvez par exemple ajouter des titres, des images, des listes et encore d'autres choses dans vos textes.
Markdown a l'avantage d'être facile à lire aussi bien en texte que lorsqu'il est mis en page, et apprendre ses conventions ne devrait vous prendre plus que quelques minutes.
Vous ne voulez pas l'utiliser ? Pas de problème, faites comme si il n'existait pas, vous pouvez taper du texte normalement comme vous l'avez toujours fait.

Titres

Vous pouvez écrire un titre en ajoutant le symbole # au début de la ligne. Au plus vous ajoutez de # consécutifs, au plus le titre sera petit. (Vous pouvez en ajouter 5 au maximum)

{% markdown_demo %} # Titre principal ## Sous titre ##### Titre insignifiant {% end_markdown_demo %}

Citations

Pour citer un texte, il suffit de rajouter un > devant chaque ligne.

{% markdown_demo %} Comme disait Mao : > Manger c'est tricher, mais tricher, c'est gagner. {% end_markdown_demo %}

Gras et italique

Tu peux rendre ton texte en gras ou en italique en entourant la partie concernée par des * (simples pour l'italique, double pour le gras).

{% markdown_demo %} Avec *une étoile, c'est du texte en italique*, mais avec **deux étoiles, c'est du texte en gras** {% end_markdown_demo %}

Liens et images

Markdown supporte aussi les liens et images.
Un lien s'écrit de la manière suivante [Texte du lien](http://site-web.com).

{% markdown_demo %} Les points sont disponibles sur [l'université virtuelle](https://uv.ulb.ac.be) {% end_markdown_demo %}

L'ajout d'images se fait de manière assez semblable : ![titre](http://example.com/image-a-afficher.jpg) (notez le point d'exclamation au début)

{% markdown_demo %} ![logo ULB](https://www.ulb.ac.be/dre/com/docs/logo3lpbp.jpg) notre alma mater {% end_markdown_demo %}

Listes

Vous pouvez aussi ajouter des listes (à puce ou numérotées)

{% markdown_demo %} * Une puce * Une autre puce * Encore une autre puce Liste numérotée : 1. Premier élément 2. Second élément 3. Troisième élément {% end_markdown_demo %}

Formules mathématiques

Vous pouvez même insérer des formules mathématiques en utilisant la notation $\LaTeX$, encadrée par des $. La notation mathématique $\LaTeX$ est bien trop complète pour être décrite ici, mais il existe un excellent Wikilivre à ce sujet.

{% markdown_demo %} La tranformée de Fourier de $f(x)$ est $F(\theta) = \int_{-\infty}^{+\infty} f(x)e^{i\theta x} \mathrm dx$ La factorielle est définie comme $n! = \begin{cases} 1 & n \leq 1 \\ n\times(n-1)! & n \gt 1 \end{cases}$ {% end_markdown_demo %}

Code

Pour nos amis informaticiens qui voudraient discuter sur du code, il est aussi possible d'insérer des bouts de codes qui seront mis en forme. Il suffit de le mettre en forme avec l'annotation ```LANGAGE ```, où LANGAGE est le nom du langage dans lequel le code est écrit.
